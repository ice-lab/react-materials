# react-materials

飞冰（ICE）官方 React 物料（模板/区块/组件），基于 Fusion 组件和 ice-scripts 工程工具。

## 使用

对应的物料数据：`https://ice.alicdn.com/assets/react-materials.json`，可以通过 Iceworks 的自定义物料功能使用。

## 如何贡献

- Fork 仓库，修改代码
- 提交 PR

## 开发指南

### 环境准备

```bash
$ npm install
```

### 开发调试

```bash
$ cd blocks/Example
$ npm install
$ npm run start
$ npm run build
```
